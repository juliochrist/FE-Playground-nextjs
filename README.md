# FrontEnd Playground

Build Once. Reuse Everywhere.

Frontend Playground is a Next.js 15 design system platform and component laboratory for Julio products including TradeIntel, SmartPOS AI, ClientPulse CRM, Life OS, and future SaaS applications.

## Stack

- Next.js 15 App Router
- TypeScript
- Tailwind CSS v4
- shadcn/ui-style primitives with Radix UI foundations
- next-themes
- React Hook Form and Zod
- Zustand
- Recharts
- Lucide React
- Framer Motion

## Run

```bash
npm install
npm run dev
```
